configuration sizes type units

configuration

$ratios variable

Defined your own ratios, or aliases for built-in ratios.

$ratios: () !default;
$ratios: (
  'line-height': 1 / 3,
  'headings': 'linear',
  'golden': 1.61803399,
);

$default-units variable

Defined the default units to use when accessing sizes.

$default-units: 'rem' !default;

$sizes variable

Defined a palette of common sizes to be used across your project, in the format "name: size" or "name: basis (ratio: multiplier)". If your map includes a value for root, it will be used in relative-size unit conversions.

$sizes: (
  'root': $_BROWSER-DEFAULT-FONT-SIZE,
  'text': 'root',
  'rhythm': 'text' ('fifth': 1),

  'h1': 'text' ('linear': 4),
  'h2': 'text' ('linear': 3),
  'h3': 'text' ('linear': 2),
) !default;
